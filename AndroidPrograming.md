# Android

### 스마트폰의 개요

+ 스마트폰 

  + 통화기능 + 컴퓨터 + 다양한 기능 내장(MP3, 카메라 DMB, GPS등).
<br/>

+ 스마트폰의 역사

  + 1992년 IBM사의 사이먼
  
  + 1996년 노키아 9000
  
  + 2002년 마이크로소프트 포켓PC
  
  + 2007년 아이폰
  
  + 2008년 안드로이드 폰
  
  + 2010년 윈도폰7
  <br/>

+ 스마트폰의 운영체제

  + 2020년 2분기 기준 세계시장 점유율 : 안드로이드 74%, 아이폰 25%, 윈도폰 0.01%
  <br/>
